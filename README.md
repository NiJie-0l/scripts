# scripts

Collection of utility scripts for macOS development workflow.

## Scripts

| Script | Description |
|--------|-------------|
| `feishu-sync.sh` | Feishu docs ↔ Git sync (in feishu-sync repo) |
| `workspace-init.sh` | Initialize new project in workspace |
| `repo-status.sh` | Check status of all workspace repos |

## Usage

```bash
# Make executable
chmod +x scripts/*.sh

# Run
./scripts/workspace-init.sh my-project public
```
